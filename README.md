# 🚀 Mi Portafolio Personal & Galería de Mascotas

Este es un proyecto web estático desarrollado como parte de un desafío de estructura y diseño web. Consiste en un portafolio profesional integrado con una sección dedicada a mis mascotas, utilizando **HTML5** y **CSS3** puro.

## 📂 Estructura del Proyecto

El proyecto se organiza en una carpeta raíz llamada `portafolio/` con la siguiente estructura:

- `index.html`: Página principal que contiene mi perfil, proyectos y formulario de contacto.
- `mascotas.html`: Sección secundaria con una galería de imágenes y descripciones.
- `styles.css`: Hoja de estilos compartida que define la identidad visual del sitio.

## 🛠️ Características Principales

### 1. Portafolio (`index.html`)
- **Header & Navegación:** Acceso rápido a las secciones y enlace directo a la página de mascotas.
- **Sobre Mí:** Una breve descripción personal.
- **Proyectos:** Lista organizada de trabajos realizados.
- **Contacto:** Formulario funcional (maquetado) para recibir mensajes.

### 2. Sección de Mascotas (`mascotas.html`)
- **Galería Visual:** Presentación de tres mascotas mediante el uso de etiquetas `<article>` e `<img>`.
- **Descripciones:** Detalles únicos de cada compañero animal.
- **Navegación de Retorno:** Enlace optimizado para volver fácilmente al inicio.

### 3. Diseño y Estilos (`styles.css`)
- **Modelo de Caja:** Uso estratégico de `padding`, `margin` y `display` para un layout limpio.
- **Interactividad:** Efectos `hover` en enlaces para mejorar la experiencia de usuario (UX).
- **Estética Moderna:** Bordes redondeados y sombras (`box-shadow`) en imágenes para un acabado profesional.
- **Tipografía y Color:** Selección de fuentes legibles y paleta de colores con contraste adecuado.

## 🚀 Cómo visualizar el proyecto

1. Descarga o clona este repositorio.
2. Asegúrate de que todos los archivos estén en la misma carpeta.
3. Abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Firefox, Edge, etc.).
